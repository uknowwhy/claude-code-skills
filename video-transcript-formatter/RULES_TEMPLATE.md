## 二、重复词汇类（需看上下文）

### "说"的重复
```regex
说这个 → （删除）
说你 → （删除）
说是一个 → 是
说是 → 是
说有一个 → 有
```

### "这个/那个"的重复
连续出现时，保留第一个，删除后续：
```regex
这个这个 → 这个
那个那个 → 那个
这个……那个…… → 删减到最少
```

### "咱们/我们"的重复
```regex
我们……我们…… → 删除第二个
咱们……我们…… → 统一为"我们"
```

## 六、标点符号类

### 中英文标点统一
```regex
。。→ 。
，，→ ，
!.→ ！
.，→ 。（中文内容用中文标点）
，，→ ，（删除重复标点）
，，→ ，
```

### 中英混排标点
- 中文内容中穿插英文：**使用中文标点**
- 例如："使用GLM 4.7模型，"（不是"使用GLM 4.7 model,"）

### 断句修正
```regex
然后他会对于。跟你的 → 然后他会处理跟你的
我在这里。来打一个 → 我在这里输入
```

## 九、保留的语气词

以下情况**不删除**，保持自然口语感：
- 句首：当然、那么、所以、因此
- 设问：对不对呢？你知道吗？
- 反问：是不是？对吧？
- 强调：特别是、尤其是、确实

## 十、需要人工判断的场景

1. **指代修正**："它"→"他"/"她"（需看上下文指人还是物）
2. **语序调整**：过长的句子需要拆分或调整语序
3. **逻辑连接**：添加或调整连接词，让行文更流畅
4. **专业术语**：确保术语一致性，不误删
5. **专有名词**：人名、书名、机构名保持准确

## 十一、处理流程

1. **第一遍**：机械替换（规则1-8）
2. **第二遍**：人工精修（规则9-10）
3. **第三遍**：通读校对
